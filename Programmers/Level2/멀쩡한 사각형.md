### solve 1

`Solving Strategies`의 코드화

### solve 1 Code

```js
function GCD(a, b) {
  if (b === 0) {
    return a;
  }
  return GCD(b, a % b);
}

function solution(w, h) {
  const gcd = GCD(w, h);
  return w * h - (w + h - gcd);
}
```
